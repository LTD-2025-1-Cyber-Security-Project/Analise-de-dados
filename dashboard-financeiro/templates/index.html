Instruções de Uso

Upload de Dados

  1. Acesse a página de upload através do menu lateral ou do botão acima.
  2. Selecione seu arquivo CSV contendo os dados financeiros.
  3. Certifique-se de que o arquivo tenha as colunas necessárias como: Entidade, Credor, Tipo Pessoa, etc.
  4. Clique em "Enviar" para processar o arquivo.

Análise dos Dados

  1. Após o upload, você será redirecionado para o dashboard automaticamente.
  2. Explore os diferentes gráficos e visualizações interativas.
  3. Utilize os filtros disponíveis para refinar sua análise.
  4. Para mais detalhes, passe o mouse sobre os elementos gráficos.

Exportação

  1. Para exportar os dados analisados, use o botão "Exportar" no menu.
  2. Os dados serão exportados em formato Excel (.xlsx) com formatação automática.
  3. O arquivo exportado conterá todos os dados processados para uso em outras ferramentas.

Ajuda

Se precisar de ajuda adicional ou tiver dúvidas sobre o funcionamento da ferramenta, consulte a documentação completa ou entre em contato com o suporte.

Formato esperado do CSV:
O sistema espera um arquivo CSV com delimitador ponto-e-vírgula (;) contendo as seguintes colunas: 

"Entidade";"Credor";"Tipo Pessoa";"Numero Ordem de Pagamento";
"Empenho";"Vinculo";"Data";"Valor Pago";"Numero";"Ano"
